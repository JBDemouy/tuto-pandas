# tuto-pandas

![Logo de pandas](http://pandas.pydata.org/_static/pandas_logo.png)

Tutoriel d'introduction à pandas pour journalistes de données ou chercheurs en humanités numériques.<br>
:panda_face: :panda_face: :panda_face:
-----

Le [carnet jupyter](kung-fu-panda.ipynb) inclus dans ce répertoire vous fait une introduction, pas à pas, de [pandas](http://pandas.pydata.org/) ([documentation](http://pandas.pydata.org/pandas-docs/stable/index.html)), bibliothèque python permettant d'analyser et d'interroger rapidement des données. Pandas est particulièrement utile lorsqu'on a des fichiers volumineux qu'un tableur comme *Excel*, *Calc* ou *Google Sheets* a du mal à avaler.
